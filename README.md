Eye Fatigue Detection System is a computer vision and deep learning project designed to detect signs of eye fatigue by analyzing facial image features. The system uses Convolutional Neural Networks (CNNs) trained on extracted eye and mouth regions to classify fatigue-related patterns. A Flask-based web application provides an interactive interface for running predictions and visualizing results.

Key Features
Eye fatigue detection using deep learning models
Facial feature extraction and preprocessing
CNN-based fatigue classification
Flask-powered web interface
Automated prediction workflow
Model evaluation and performance analysis
